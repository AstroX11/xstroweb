DEPLOYMENTS

Heroku Deploy

  1. Create an Heroku Account
  2. Setup Environment Variables:
    SESSION_ID, BOT_INFO, SUDO, STICKER_PACK, WARN_COUNT, and TIME_ZONE.

Once you have understood the above steps, click the button below.

Koyeb Deploy

  1. Create an Koyeb Account
  2. Setup Environment Variables:
    SESSION_ID, BOT_INFO, SUDO, STICKER_PACK, WARN_COUNT, and TIME_ZONE.

Once you have understood the above steps, click the button below.

Render Deploy

  1. Create a Render Account
  2. Setup Environment Variables:
    SESSION_ID, BOT_INFO, SUDO, STICKER_PACK, WARN_COUNT, and TIME_ZONE.
  3. Deploy Blueprint and wait for the bot to start.
  4. Copy the deployed URL.
  5. To ensure uptime, set up a monitor using Better Stack:
    SignUp OR Login
  6. In Better Stack, create a monitor with your deployed URL (pings every 5 minutes).

Once you have understood the above steps, click the button below.

Local Deployment (Windows/Linux/macOS)

  1. Install Node.js
    Windows:
    Download Node.js and install it. Verify installation: 
    node -v
npm -v
    Linux:
    curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt install -y nodejs
node -v
npm -v
    macOS:
    brew install node
node -v
npm -v
  2. Install FFmpeg
    Windows:
    Download FFmpeg, extract it, and add the bin folder to your PATH. Verify installation: 
    ffmpeg -version
    Linux:
    sudo apt update && sudo apt install ffmpeg
ffmpeg -version
    macOS:
    brew install ffmpeg
ffmpeg -version
  3. Install Yarn
    Install globally: 
    npm install -g yarn
yarn -v
  4. Install Git
    Windows:
    Download Git and install it. Verify installation: 
    git --version
    Linux:
    sudo apt update && sudo apt install git
git --version
    macOS:
    brew install git
git --version
  5. Setup and Run
    Open a terminal and execute the following commands: 
    git clone https://github.com/AstroX11/Xstro.git
cd Xstro
yarn install
npm start
